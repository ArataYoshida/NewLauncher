# NewLauncher

## 日本語

NewLauncher は NewEngine / NewEditor のインストール、更新、プロジェクト起動を行う Windows 向けランチャーです。

主な機能:

- GitHub Releases から NewEngine の最新版と過去バージョンを取得
- インストール済みバージョンの表示
- ダウンロード進捗とパーセント表示
- 起動時の NewLauncher 自動アップデート
- プロジェクト作成と Editor 起動

NewEngine / NewEditor のソースコードはこのリポジトリには含まれません。

## English

NewLauncher is the Windows launcher for installing and updating NewEngine / NewEditor and opening projects.

Features:

- Fetch the latest and older NewEngine versions from GitHub Releases
- Show installed versions
- Display download progress and percentage
- Auto-update NewLauncher on startup
- Create projects and launch the Editor

This repository does not contain the NewEngine or NewEditor source code.
